Resumo:

Este trabalho envolve a criação de um servidor Node.js para uma Aplicação de Registo de Teses.

A parte mais importante deste trabalho foi a compreensão do funcionamento de formulários HTML e as diferenças entre os vários métodos existentes, como GET e POST.

Como parte deste trabalho usamos a ferramenta Pug para desenvolvimento das páginas HTML.

Para guardar as teses registadas usamos um ficheiro de formato JSON como base de dados local.
